PAUTAS DE ACCESIBILIDAD PARA EL CONTENIDO WEB

Principio 1. Perceptible

El contenido de la página debe ser legible y funcional cuando se dobla el tamaño del texto. Se recomienda comprobar que el texto puede ser ampliado hasta un 200%, además de ofrecer la alternativa de aumentar o disminuir el tamaño del texto ofreciendo al usuario herramientas de accesibilidad.

  • Elementos como imagen, video y audio cuentan con una alternativa textual.
  • Si existen elementos de audio pre grabado, se debe considerar una transcripción descriptiva para ellos.
  • Si existen videos pre grabados en el sitio, se debe ofrecer subtítulos para ellos.
